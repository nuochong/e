# 解决方案如下

# 方案一（建议使用）

下载 python2.7 安装即可：https://www.python.org/download/releases/2.7/ (opens new window)

ps：安装完成以后要添加环境变量。 编辑环境变量 安装完成以后关闭 cmd ，再次重新执行 npm/yarn install。

# 方案二

管理员权限打开执行 cmd，执行以下命令：

npm install --global --production windows-build-tools
